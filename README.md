## ⚠️ 주의 사항
daisyUI를 사용하기 위해서는 nodejs가 필요합니다. nodejs가 설치돼있지 않은 분은 설치해주세요.
<br>
<br>
원격 저장소를 받아온 뒤 프로젝트 루트 디렉토리에서 아래 명령어를 실행하세요.
<br>
```bash
npm install  
npm run build:css
```
